# Tic Tac Toe (Tkinter GUI) ❌⭕  

A simple Tic Tac Toe game built with Python and Tkinter.  
Players can click on the grid to place X or O, and the game automatically detects wins, losses, or draws.

## Features
- Clean and simple Tkinter interface
- Click-based gameplay (no console input)
- Automatic win / draw detection
- Game reset after each round

## Why I built it
To learn GUI programming with Tkinter and practise implementing game logic in Python.
